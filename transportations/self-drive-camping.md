# 自驾露营越野车 (Self-Drive Camping 4x4)

## 🏕️ 车型介绍

适合有丰富越野及露营经验的冒险者。我们提供配备完整露营装备的 **Toyota Land Cruiser** 或 **Land Rover Defender**。

### 标准配置

#### 车辆装备
*   **车顶帐篷**：2-3人容量，带床垫及睡袋
*   **副油箱**：总油箱容量200L+，续航超过1000公里
*   **绞盘及脱困装备**：高举千斤顶、拖车绳、沙板
*   **GPS导航系统**：预装公园路线及营地坐标
*   **车载冰箱/冰柜**：60L容量

#### 露营装备
*   折叠桌椅（4人份）
*   炉具及燃气罐
*   炊具套装（锅、碗、餐具等）
*   10L水箱 x 2
*   应急食品及饮用水
*   营地照明（LED灯 + 太阳能充电板）

### 价格参考

*   **车辆租金**：$180-250 / 天
*   **露营装备包**：$50 / 天
*   **GPS及卫星电话租赁**：$15 / 天
*   **押金**：$2000（刷卡预授权）

## ⚠️ 租车要求

*   国际驾照 + 原始驾照
*   驾龄至少3年
*   年龄25岁以上
*   签署免责协议
*   强烈建议购买全险（含道路救援）

## 📋 推荐路线

### 北部环线自驾（7-10天）
阿鲁沙 → 塔兰吉雷 → 曼雅拉湖 → 恩戈罗恩戈罗 → 塞伦盖蒂 → 返回阿鲁沙

### 南部冒险线（12-14天）
达累斯萨拉姆 → 米库米 → 鲁阿哈 → 尼雷尔 → 返回达累斯萨拉姆

> ⚠️ **重要提示**：
> - 西部环线（卡塔维、马哈勒）不建议自驾（道路极差且无加油站）
> - 雨季（3-5月）不建议自驾游猎
> - 必须提前预订营地（公园内营地不接受无预约到访）
